## How to run

create a virtual environment

`python -m venv .venv`

activate the virtual envinronment: 

on linux  :  `source .venv/bin/activate`
on windows: `.venv/bin/activate.ps1/`

run the server

`cd app && python manage.py runserver`

default port is 8000, which can be changed by adjusting the `runserver` command

`python manage.py runserver wxyz`
